#### 网页基本信息

1. 标签  单个标签，自闭合标签，闭合标签

```HTML
<!DOCTYPE html>

<!--总标签 html-->
<html lang="en">

<!--head标签 代表网页头部 -->
<head>
    <!--meta描述性标签，他用来描述我们网站的一些信息 -->
    <!--一般用来做SEO-->
    <meta charset="UTF-8">
    <meta name = 'keywords' content="网页">
    
    <!--title网页标题-->
    <title>Title</title>
</head>

<!--body标签 网页主体-->
<body>

</body>
```
