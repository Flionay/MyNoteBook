## 二、将硬盘分区。

1、当硬盘小于等于2T时，可以用 `fdisk` 

```
fdisk /dev/sdb
1、查看帮助。
输入：m
2、新建分区。
输入：n
3、创建逻辑分区
输入：p
4、输入分区号以及指定分区大小
依照提示，回车表示默认。
5、检查分区情况（此时还未执行分区操作）
Command（m for help）：p 
6、保存退出
Command（m for help）：w
```

2、当硬盘大于2T时，用parted命令。

```
parted /dev/sdb   (用part命令对3T硬盘进行分区处理）
mklabel gpt       (用gpt格式可以将3TB弄在一个分区里)
unit TB           (设置单位为TB)
mkpart primary 0 3 (设置为一个主分区,大小为3TB，开始是0，结束是3）
print              (显示设置的分区大小）
quit               (退出parted程序)
```

## 三、格式化分区。

```
mkfs.ext4 /dev/sdb1
```

## 四、将硬盘挂载到文件夹下。

1、手动挂载。

新建一个文件夹：mkdir /diskb

挂载：mount /dev/sdb /diskb

2、开机自动挂载。

输入：vi /etc/fstab

在最后加入：

```
/dev/sdb    /diska    ext4    defaults    1    1
```

> 最后要注意更改磁盘的读取写入权限。
